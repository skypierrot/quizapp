### 네트워크 구성도

```
사용자 → quiz.onthe.duckdns.org → Nginx Proxy Manager → http://quizapp:3000
                                                            |
                               ngnet Docker 네트워크를 통한 통신
                                     |             |
                                PostgreSQL      기타 서비스
```

### 장점 및 고려사항

- **보안성**: 내부 서비스(quizapp)가 직접 외부에 노출되지 않음
- **유연성**: 내부 포트 또는 서비스 변경 시 Nginx 설정만 변경하면 됨
